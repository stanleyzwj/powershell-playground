# Day 1 - PowerShell + Git Learning Journey 🚀

**Date:** 2026-05-02

## What I learned today

### PowerShell
- `Get-Process` — lists all running processes
- `Get-Date` — shows current date/time  
- `$env:USERNAME` — environment variables
- Pipeline `|` — chains commands together
- `Sort-Object`, `Select-Object`, `Where-Object`
- `@"..."@` — Here-String for multi-line text

### Git & GitHub
- `git status` — see what changed
- `git add .` — stage all changes  
- `git commit -m` — save a snapshot
- `git push` — upload to GitHub
- GitHub PAT — authentication token

## Key Insights
